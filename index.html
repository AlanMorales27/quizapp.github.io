Capacitación A-CDM

Recurso interactivo

Algunas reglas que debes conocer

  1. 1. Solo tienes 45 segundos para responder cada pregunta.

  2. 2. Solamente podrás seleccionar una respuesta, así que asegúrate de ella.

  3. 3. Si se termina el tiempo, no podrás seleccionar una respuesta.

  4. 4. Durante el juego, no podrás salir del quiz.

  5. 5. Si tu respuesta es correcta ganarás puntos.
